# 统一使用关键字 function 实现计划

## 目标
统一 X 语言代码库中函数定义的关键字使用，从 `fun` 改为 `function`，确保所有代码和文档一致使用完整的 `function` 关键字。

## 现状分析

### 词法分析器
- **文件**: `compiler/x-lexer/src/lib.rs`
- **现状**: 词法分析器目前同时支持 `function` 和 `fun` 关键字，都映射到 `Token::Function`
- **位置**: 第 146-147 行
  ```rust
  "function" => Ok(Token::Function),
  "fun" => Ok(Token::Function),  // 需要移除这一行
  ```

### 使用 `fun` 关键字的文件

1. **标准库文件**:
   - `library/stdlib/sys.x` - 多处使用 `fun` 关键字定义函数

2. **测试文件**:
   - `test/simple_test.x` - 使用 `fun` 关键字定义 main 函数

3. **文档**:
   - 文档中已经主要使用 `function` 关键字，但需要确认所有文档都一致

## 实现步骤

### 1. 修改词法分析器
- **文件**: `compiler/x-lexer/src/lib.rs`
- **操作**: 移除 `"fun" => Ok(Token::Function),` 这一行
- **目的**: 只允许使用 `function` 关键字，不再支持 `fun`

### 2. 更新标准库文件
- **文件**: `library/stdlib/sys.x`
- **操作**: 将所有 `fun` 关键字替换为 `function`
- **涉及函数**: 所有使用 `fun` 定义的函数

### 3. 更新测试文件
- **文件**: `test/simple_test.x`
- **操作**: 将 `fun main() {` 替换为 `function main() {`

### 4. 检查其他文件
- **操作**: 搜索整个代码库中所有使用 `fun` 关键字的地方
- **工具**: 使用 grep 或类似工具搜索 `fun` 关键字
- **目的**: 确保没有遗漏任何使用 `fun` 的地方

### 5. 运行测试
- **操作**: 运行编译器的测试套件
- **目的**: 确保所有更改都正常工作，没有引入任何错误

## 预期结果

1. 词法分析器不再支持 `fun` 关键字，只接受 `function`
