# X-Lang C23 后端

实现流水线：**X → X IR → C23 → GCC/Clang → .o**

## 概述

这是 X-Lang 的第一个编译后端，使用 C23 作为中间目标语言。

## 流水线架构

```
┌─────────────┐
│  X 源码      │  (.x 文件)
└──────┬──────┘
       │ 1. 解析
       ↓
┌─────────────┐
│   AST       │  (抽象语法树)
└──────┬──────┘
       │ 2. Lowering
       ↓
┌─────────────┐
│   X IR      │  (X 中间表示)
└──────┬──────┘
       │ 3. C 代码生成
       ↓
┌─────────────┐
│  C23 源码   │  (.c 文件)
└──────┬──────┘
       │ 4. 编译 (GCC/Clang/MSVC)
       ↓
┌─────────────┐
│  目标文件   │  (.o / .obj)
└─────────────┘
```

## 目录结构

## X IR (X Intermediate Representation)

X IR 是一个类 C 的中间表示，设计用于方便地 lowering 到 C23。

## 使用方法

### 1. 作为库使用

```rust
use x_codegen::*;

// 创建 C 后端配置
let config = CBackendConfig {
    output_dir: Some(PathBuf::from("output")),
    optimize: true,
    debug_info: true,
    c_standard: CStandard::C23,
    compiler: CCompiler::Gcc,
};

// 创建后端
let mut backend = CBackend::new(config);

// 从 AST 生成 C 代码
let c_code = backend.generate_from_ast(&ast_program)?;

// 或从 X IR 生成
let c_code = backend.generate_from_xir(&xir_program)?;

// 编译到 .o
backend.compile_c_code(&c_code, &PathBuf::from("output.o"))?;
```

### 2. 使用 CLI

```bash
# 编译到 C 代码
cargo run -- compile hello.x --emit c -o hello.c

# 编译到 .o (需要 C 编译器)
cargo run -- compile hello.x -o hello.o

# 完整管道：X → C → 可执行
cargo run -- compile hello.x -o hello
```

## 示例

### Hello World

**X 源码** (`hello.x`):
```x
fun main() {
  print("Hello, World!")
}
```

**生成的 C23** (`hello.c`):
```c
// Generated by X-Lang C23 backend
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

extern int printf(const char*);

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

**编译和运行**:
```bash
gcc -std=c2x -o hello hello.c
./hello
# 输出: Hello, World!
```

### Fibonacci

**X 源码** (`fib.x`):
```x
fun fib(n) {
  if n <= 1 {
    return n
  }
  return fib(n - 1) + fib(n - 2)
}

fun main() {
  let i = 0
  while i < 10 {
    print("fib(")
    print(i)
    print(") = ")
    print(fib(i))
    print("\n")
    i = i + 1
  }
}
```

**生成的 C23**:
```c
// Generated by X-Lang C23 backend
#include <stdio.h>
#include <stdlib.h>
#include <stdbool.h>

extern int printf(const char*);

int fib(int n) {
    if (n <= 1) {
        return n;
    }
    return fib(n - 1) + fib(n - 2);
}

int main(void) {
    int i = 0;
    while (i < 10) {
        printf("fib(");
        printf("%d", i);
        printf(") = ");
        printf("%d", fib(i));
        printf("\n");
        i = i + 1;
    }
    return 0;
}
```

## C 标准支持

| 标准 | GCC/Clang | MSVC |
|------|-----------|------|
| C23 | `-std=c2x` | 有限支持 |
| C17 | `-std=c17` | `/std:c17` |
| C11 | `-std=c11` | `/std:c11` |
| C99 | `-std=c99` | `/std:c99` |

默认使用 C23，但可以配置为其他标准。

## 编译器支持

- **GCC**: 推荐，完整的 C23 支持
- **Clang**: 优秀的 C23 支持
- **MSVC**: 有限的 C23 支持，推荐使用 C17

## 优化级别

| 级别 | GCC/Clang | MSVC | 说明 |
|------|-----------|------|------|
| O0 | `-O0` | `/Od` | 无优化，调试 |
| O1 | `-O1` | `/O1` | 基础优化 |
| O2 | `-O2` | `/O2` | 推荐优化 |
| O3 | `-O3` | `/O2` | 激进优化 |
| Os | `-Os` | `/O1` | 优化大小 |
