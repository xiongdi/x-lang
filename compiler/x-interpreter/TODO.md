# x-interpreter TODO List

## 项目概览

x-interpreter 是 X 语言的树遍历解释器，负责直接执行 X 语言程序的抽象语法树。

**整体完成度**：约 75%

**当前覆盖范围**：提升中（以 `cargo llvm-cov -p x-interpreter --tests` 为准）
