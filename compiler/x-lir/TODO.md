# x-lir TODO

## 项目概览

x-lir 是 X 语言的低层中间表示（Low-level Intermediate Representation），也称为 XIR。它是所有代码生成后端的统一输入。

## 当前状态

- **完成度**: ~65%
- **核心功能**: LIR 数据结构、MIR → LIR 降级
- **集成状态**: 与 x-mir 和各代码生成后端集成完成
