X语言安装指南

系统要求

X语言支持以下操作系统：

  • Windows 10 或更高版本
  • macOS 10.15 或更高版本
  • Linux（Ubuntu 20.04+、Debian 10+、Fedora 32+ 等）

硬件要求：

  • 至少 2GB 内存
  • 至少 10GB 磁盘空间
  • 支持 64 位架构的处理器

安装方法

Windows

  1. 访问 X语言发布页面 下载最新的 Windows 安装包（.msi 文件）。
  2. 双击安装包，按照安装向导的提示完成安装。
  3. 安装完成后，打开命令提示符或 PowerShell，输入以下命令验证安装：
x --version

如果显示版本信息，则表示安装成功。

macOS

  1. 访问 X语言发布页面 下载最新的 macOS 安装包（.pkg 文件）。
  2. 双击安装包，按照安装向导的提示完成安装。
  3. 安装完成后，打开终端，输入以下命令验证安装：
x --version

如果显示版本信息，则表示安装成功。

Linux

使用包管理器安装

Ubuntu/Debian：

sudo apt update
sudo apt install x-lang

Fedora：

sudo dnf install x-lang

使用安装脚本安装

  1. 打开终端，输入以下命令：
curl -fsSL https://raw.githubusercontent.com/xiongdi/x-lang/main/install.sh | bash
  1. 安装完成后，输入以下命令验证安装：
x --version

如果显示版本信息，则表示安装成功。

从源码构建

如果您需要从源码构建X语言，可以按照以下步骤操作：

  1. 安装依赖：
    • Rust 1.60 或更高版本
    • Git
    • CMake（可选，用于某些依赖）
  2. 克隆仓库：
git clone https://github.com/xiongdi/x-lang.git
cd x-lang
  1. 构建并安装：
cargo install --path .
  1. 验证安装：
x --version

环境变量配置

安装完成后，X语言的可执行文件应该已经添加到系统路径中。如果遇到 "command not found" 错误，可以手动添加环境变量：

Windows

  1. 右键点击「此电脑」，选择「属性」。
  2. 点击「高级系统设置」。
  3. 点击「环境变量」。
  4. 在「系统变量」中找到「Path」，点击「编辑」。
  5. 添加X语言安装目录（默认为 C:\Program Files\X-Lang\bin）。
  6. 点击「确定」保存更改。

macOS/Linux

  1. 打开终端，编辑 ~/.bashrc 或 ~/.zshrc 文件：
nano ~/.bashrc
  1. 在文件末尾添加：
export PATH="$HOME/.cargo/bin:$PATH"
  1. 保存并退出编辑器，然后运行：
source ~/.bashrc

验证安装

安装完成后，您可以通过以下方式验证X语言是否安装成功：

x --version

如果显示版本信息，则表示安装成功。您也可以尝试运行一个简单的X语言程序：

echo 'fn main() { print("Hello, World!"); }' > hello.x
x run hello.x

如果输出 "Hello, World!"，则表示X语言可以正常运行。

卸载X语言

Windows

  1. 打开「控制面板」->「程序和功能」。
  2. 找到「X语言」，点击「卸载」。
  3. 按照卸载向导的提示完成卸载。

macOS

  1. 打开「Finder」->「应用程序」。
  2. 找到「X语言」，拖拽到「废纸篓」。
  3. 清空「废纸篓」。

Linux

使用包管理器卸载：

# Ubuntu/Debian
sudo apt remove x-lang

# Fedora
sudo dnf remove x-lang

从源码安装的卸载：

cargo uninstall x

常见问题

Q: 安装过程中出现错误怎么办？

A: 请检查是否满足系统要求，确保网络连接正常。如果问题仍然存在，请查看 GitHub Issues 或提交新的问题。

Q: 如何更新X语言？

A: 可以使用与安装相同的方法更新X语言，或者在命令行中运行：

x upgrade

Q: 安装后无法找到x命令怎么办？

A: 请检查环境变量配置是否正确，确保X语言的安装目录已添加到系统路径中。
